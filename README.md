# LaserCache
前端一个轻型的缓存工具，防止内存“侧漏”
## 简介

## 安装下载
- 下载地址 https://github.com/xiangcheng0618/LaserCache/releases/tag/v0.0.1
- `npm i LaserCache`
- CDN http://unpkg.com/fast-cache-npm/release/bundle.js
简单接入的文档

## 快速使用
```js
var LC=window.LaserCache;
var cache=new LC();
cache.set('a',100);
alert(cache.get('a'));
```
